Overview

Blast away deductions and solve the magical mystery in this danmaku-detective game!

A game made for CMPUT 250 in the Fall 2020 term over about 4 months. I created all the art and planned the overall design of the game.

Developing Assets Quickly

I was the only artist on Star Apprentice, so in order to complete the necessary assets it was important to simplify and minimize the workload. My strategies for doing so revolve around prioritizing assets by importance.

I applied Limited Animation techniques to the graphics of the game as a major way to get things done quickly. The strategy is to animate only what is necessary, and use as few frames as possible doing so. Minnie’s walk cycle is a great example of this as it uses only 6 frames. Though the game runs at 60 FPS each of these frames is held for 8 frames (meaning the animation is played at 7.5 FPS). If we were to animate this at the full 60 FPS we would need 48 frames, which is clearly not feasible.

Though smaller sprites were quick to make, there were a few large sprites which needed strategy to complete quickly. One such example is the background in the “Confrontation” segments. Though the game is made with pixel art, this is one component that was not made as pixel art. Instead it was made by placing some planes in a Unity scene and taking screenshots while controlling the scrolling of the planes. The color palette was then restricted afterward to help it fit with the other assets.

The character portraits were also an area where I decided to spend more time on because it was important to communicating each character in more detail. The small sprite is created first to establish the important aspects of the design. To start the portrait, I posed a 3d model which would be used as reference. I then drew the details on top of this reference.
